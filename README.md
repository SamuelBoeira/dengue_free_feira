# Dengue Free Feira
Um sistema de armazenamento de casos da dengue nos bairros da cidade de Feira de Santana que procura facilitar a vida dos agentes de saúde com o grande aumento de contaminações na cidade. 
  O programa armazena os dados de cada bairro por data, cada bairro contendo seu número de habitantes, casos suspeitos, confirmados e negativos, tudo isso sendo armazenado em um arquivo CSV. Com essas informações os agentes de saúde podem gerenciar os dados já obtidos, fazendo buscas inteligentes com filtros por nome de bairro e data específicas, para atualizar o software oferece a opção de atualização, em que é possível adicionar novos casos suspeitos, negativos e positivos para o dia posterior ao armazenado. Além disso o programa Dengue Free Feira também entrega métricas para o usuário, fazendo comparação de casos entre duas datas e dados estatísticos com o decorrer das atualizações.


OBS:
> o arquivo com o número de casos "entrada.csv" precisa estar na mesma parta para o programa rodar bem.

> o programa usa a biblioteca tabulate para fins de organização, por isso é necessário ter instalado em sua máquina.
